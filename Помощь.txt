Настройки конфига:
 Для добавления ещё одного сообщения в конфиг, код ниже.
 КОД ДЛЯ ВСТАВКИ: (Скопируйте, на 11 строчке после } сделайте пробел и вставьте этот код)
    },{
        "active": true,
        "channelID": "АЙДИ ЧАТА",
        "content": "Вторая команда",
        "interval": 2000,
        "inventory": {
        	"say": { "active": true, "count": 8568119585, "content": "Мой текст #2, 3, 4 и т.д" }
          }
 "token" - твой дискорд токен, или же токен бота, как его найти я написал в самом верху.
 "channelID" - айди чата (пкм по чату и нажмите на копировать айди, вставьте его в этот параметр )
 "content" - Название твоей команды, это сделано для того чтобы ты мог их различать
 "interval" - интервал отправки сообщения в милисекундах
 "say" - отправка сообщения, другие параметры кроме него не будут работать.
 "active" - true/false, включение параметра или выключение его
 "count" - количество сообщений, выставлять его в пределах node.js
 "content": "text" - сообщение которое будет отправлятся с заданным количеством и интервалом.
